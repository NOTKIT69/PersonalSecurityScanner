# 🛡️ Personal Security Scanner

A modern Python-based security scanner that performs website security checks from a clean terminal interface.

## ✨ Features

- 🔍 Security Header Analysis
- 🔒 SSL Certificate Checker
- 🌐 Website Information
- 📊 Beautiful Terminal UI
- 📄 HTML Report Generation
- 📜 Scan History
- 🎨 Theme Support
- ⚡ Fast & Lightweight

## 📂 Project Structure

```text
PersonalSecurityScanner/
├── assets/
├── reports/
├── main.py
├── scanner.py
├── ssl_checker.py
├── headers.py
├── report.py
├── history.py
├── settings.py
├── themes.py
├── ui.py
├── requirements.txt
└── README.md
```

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/PersonalSecurityScanner.git
```

Enter the project folder:

```bash
cd PersonalSecurityScanner
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## ▶️ Usage

```bash
python main.py
```

## 📄 Reports

Generated reports are saved inside the `reports/` directory.

- Python 3.10+
- pip
